# Ioto API

The Ioto library provides some utility routines and cloud helper functions.

Most of the Ioto API is provided by the following component libraries:

* [Crypt](crypt.md)
* [Database](db.md)
* [Embedded Web Server](web.md)
* [JSON parser](json.md)
* [MQTT client](mqtt.md)
* [Safe Runtime](r.md)
* [Url HTTP Client](url.md)
